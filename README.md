## Getting Started

### Prerequisites

- python >= 3.7.9

### Install
```sh
pip install -r requirements.txt
```

### Run
```sh
export SENTRY_BASE_URL="https://sentry.io/api/0/"
export SENTRY_AUTH_TOKEN="[REPLACE_TOKEN]"
export SENTRY_EXPORTER_ORG="[organization_slug]"
```
```sh
python exporter.py
```

### Docker

```sh
docker-compose build
```
```sh
docker-compose up -d
```
